# letstouchbase (LTB)

> A corporate dating app. Hinge's mechanics, LinkedIn's skin, real
> credentials, deadpan corporate-jargon copy.
>
> **Built for your two weeks' notice.**

## What lives where

```
apps/mobile        # Expo app (the product) — Android first, iOS-ready
apps/web           # Next.js marketing site (Cloudflare Pages)
packages/shared    # Zod schemas, taxonomies, the Corporate Jargon glossary
supabase/          # migrations, edge functions, seed data
docs/plan/         # the full build plan, chunked
```

Start with [docs/plan/README.md](docs/plan/README.md).

## The two laws

1. **Zero-cost infrastructure** — free tiers only until traction pays for more.
2. **Premium never gates features — only volume.**
